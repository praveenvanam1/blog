AZ-303: Microsoft Azure Architecture Technologies

 

Certification Info

This certification's purpose is to evaluate your advanced knowledge of azure solutions implementation and IT operations. To attempt this exam, you should be subject matter expertise (SME) in implementing solutions that run on Microsoft Azure. This exam will cover mainly the implementation of networking, identity, security, virtualization, disaster recovery, business continuity, data platform, budgeting, and governance. You should also have advanced level skills in Azure administration and experience in Azure development and DevOps.

Syllabus

The below areas will be covered in this exam. You will need to scope at least 700 out of 1000, and there is no grading or ranking system for the score you secured, the result will be either pass or fail.

 

·         Implement and monitor an Azure infrastructure (50-55%)

·         Implement management and security solutions (25-30%)

·         Implement solutions for apps (10-15%)

·         Implement and manage data platforms (10-15%)

How?

This is what I did when I start preparing for this exam, and it worked for me. It is up to you to follow this order or prepare your own.
